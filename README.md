# Анализ данных и визуализация
EDA датасета [Superstore Dataset](https://www.kaggle.com/datasets/vivek468/superstore-dataset-final).

Ноутбук содержит загрузку и предварительную обработку данных, проведение описательного и статистического анализа, создание различных типов графиков с использованием библиотеки Plotly для изучения распределений и взаимосвязей между различными переменными в наборе данных.
